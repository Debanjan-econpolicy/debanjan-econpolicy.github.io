## Log Format

Each log file uses this structure:

```

## What was done
- Bullet list of changes made

## Files changed
- List of files modified/created/deleted

## Known issues / pending work
- Anything left incomplete or flagged for follow-up

## Commit(s)
- Short hash + message
```
